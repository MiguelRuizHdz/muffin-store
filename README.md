# 🛒 Tienda Web con WhatsApp Checkout

Una aplicación web simple, genérica y directa al grano para venta de productos. Diseñada para que los clientes agreguen cosas a un carrito y el pedido se levante automáticamente mediante un mensaje de WhatsApp.

## 🔥 Características

* **Catálogo de Productos:** Lista de productos disponibles.
* **Opciones de Artículos:** Permite que el usuario elija variantes necesarias (como sabores, colores, etc).
* **Carrito de Compras:** Suma o resta cantidades y calcula el total automáticamente.
* **Levantamiento por WhatsApp:** El botón final compila la orden y envía al usuario a enviar un mensaje limpio de WhatsApp sin necesidad de pasarelas de pago.

## 💻 Stack Tecnológico

* React
* Vite (TypeScript)
* Vanilla CSS

## ⚙️ Cómo correrlo en tu computadora

1. Clona el repositorio e instala todo lo necesario:
```bash
npm install
```

2. Levanta el servidor local:
```bash
npm run dev
```
